World Bank Program Budget Analysis Using Python

This project focused on analyzing the World Bank's program budget allocation over multiple fiscal years using Python. The aim was to derive insights into how funds were distributed across sectors and regions, assess execution effectiveness, and highlight trends in budget utilization.

Skills & Tools: Python, Pandas, NumPy, Matplotlib, Seaborn, Data Cleaning, Exploratory Data Analysis (EDA), Jupyter Notebook

Key Insights:

  • Analyzed program budget data from FY19 to FY23 to identify spending trends and changes in funding priorities
  • Explored allocation of funds by sector and region, highlighting top-funded areas and underfunded sectors
  • Compared planned vs actual budget values to detect discrepancies and evaluate financial execution accuracy
  • Used visualizations to communicate insights through bar charts, heatmaps, and line plots
  • Provided data-driven recommendations to improve future budget planning and monitoring
Example Visuals:
  • Bar Charts showing yearly budget allocation by sector
  • Heatmaps representing regional budget concentration
  • Line Graphs comparing planned vs executed budgets
