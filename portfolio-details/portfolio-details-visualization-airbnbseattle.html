Seattle Airbnb Price & Revenue Analysis Using Tableau

This project analyzes Airbnb listings data in Seattle to uncover pricing patterns, bedroom distribution, and revenue performance. Using Tableau, I visualized average prices per bedroom, price distribution by ZIP code, and revenue trends across time.

Skills & Tools: Tableau, Data Cleaning, Aggregation, Filtering, Maps, Time-Series Analysis

Key Insights:

  • Listings with more bedrooms command significantly higher average prices.
  • The most common listings are 1-bedroom, followed by 2-bedroom units
  • Zip code 98119 and 98134 have the highest average prices
  • Annual revenue shows consistent growth, with a spike during mid-year and end-of-year periods
